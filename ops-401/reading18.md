# Questions?

**1. Explain CloudWatch Events to a non-technical friend.**

If you ever made a big purchase, your bank may contact you to ensure that it is not an  unauthorized charge. If you know you didn't make that purchase, you can act on it to ensure you don't lose any of your money. CloudWatch Events is similar to your bank informing you about  unauthorized charges. It will tell you real-time information changes in AWS, so you know what is going on and take action on it if someone is trying to do something. 


**2. What do CloudWatch Logs helps us achieve?**

It helps us maintain a central location for reviewing logs. By reviewing these logs, we can troubleshoot our system and applications using keywords for filtering. 

**3. What capabilities does CloudWatch Anomaly detection have?**

* Learn and model expected behavior of a metric based on prior data
* Calculate expected values and generates the Anomaly Detection band
* Create alarms based on the Anomaly Detection band
* AWS API and CloudFormation support
